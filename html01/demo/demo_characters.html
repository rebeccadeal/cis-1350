HTML Character Codes

This page demonstrates how to use HTML character codes to display an extended list of character symbols in your web page. Character symbols not found on your keyboard can be entered into your web page using either

&#code;

where code is the code number for the character, or

&char;

where char is the name assigned to the character.

To view a character symbol, enter the HTML code for the character code number or name into the input box below and click the Show button. To display a table of character symbols, select the table type from the drop-down list below and click the Show Table button. The table contains a representation of the character as it appears in your browser. The character name and the character code number appear below the symbol.

Note that not all browsers support the complete set of character code names and/or code numbers.
