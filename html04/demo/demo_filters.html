CSS3 Filter Styles

The CSS3 filter styles are used to modify the appearance of a page object by altering the color scheme of the object. You alter the color scheme by modifying properties such as the color hue, saturation, brightness, or contrast. To use a CSS3 filter, you apply the style:

filter: effect(params)

where effect is the filter function that will be applied to the object and params represents any parameters required by the filter. You should also include styles with browser extensions so that the filter will appear in all popular browsers.

To apply a CSS3 filter, select one of the filter types from the drop-down list below and enter your parameter values. The filter effect will be automatically applied to the image in the Preview box. The corresponding CSS style rule is displayed in the Style box below.
