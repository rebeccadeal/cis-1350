CSS Styles

This page demonstrates how different styles affect the appearance of web page text. Enter your sample text in the box below the style options. When you press the Tab key, the text will be copied to the Preview box. Set the styles for the preview text by entering or selecting the style value from the input boxes and then pressing the Tab key. To apply the default browser style values, enter or select default. The style selections are applied automatically to the preview text and the style code is displayed in the Style box below the Preview box.

To reset all of the styles to their default values, click the Reset Styles button below the style options.

Note that depending on your browser and browser version, you might not be able to apply text and background colors using the rgba, hsl, or hsla color value models.
